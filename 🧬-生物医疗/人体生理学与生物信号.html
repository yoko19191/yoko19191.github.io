Hint

动作电位的发生是神经细胞（如神经元）和肌肉细胞通信的基础。这一过程涉及细胞膜上离子通道的精确调控。我们从细胞水平来逐步分析这个复杂但精确的过程。

1. 静息电位的建立 🌌

  • 基础状态：在没有动作电位发生时，神经细胞处于静息状态。此时细胞内外有一定的电位差，称为静息电位，通常是负值，约-70毫伏。
  • 离子浓度差异：这是由细胞内外钠离子（Na⁺）和钾离子（K⁺）浓度差异造成的。细胞外Na⁺浓度高，而细胞内K⁺浓度高。
  • 离子泵和通道：钠钾泵（Na⁺/K⁺泵）不断地将Na⁺从细胞内泵出，同时将K⁺泵入细胞内，维持这种浓度梯度。

2. 动作电位的触发 🚀

  • 阈值：当神经细胞的膜电位由于刺激而达到一定阈值（通常是-55毫伏左右），动作电位就会被触发。
  • 离子通道打开：达到阈值时，特定的电压门控钠离子通道迅速打开，Na⁺迅速流入细胞。

3. 去极化阶段 ⚡

  • Na⁺的大量流入：当这些钠通道打开，Na⁺按照浓度梯度和电势差流入细胞内部，导致细胞内部变得更正。
  • 电位上升：这个过程称为去极化，细胞膜电位迅速上升，可达+30至+40毫伏。

4. 复极化阶段 🔙

  • 钠通道关闭，钾通道打开：随着膜电位的升高，电压门控钠通道开始关闭，电压门控钾通道打开。
  • K⁺流出：钾离子（K⁺）开始流出细胞，这有助于恢复细胞内部的负电位。

5. 过度复极化和恢复 🔄

  • 过度复极化：由于K⁺通道关闭的速度较慢，会有短暂的过度复极化阶段，膜电位可能会暂时变得比静息电位更负。
  • 离子泵的作用：最后，钠钾泵重新建立起正常的离子浓度梯度，使细胞恢复到静息电位。

🖼️ 图像化描述：

想象一条曲线：开始时在-70毫伏处，代表静息状态。突然，曲线急剧上升至+40毫伏，显示Na⁺的流入（去极化）。然后曲线下降，回到-70毫伏以下（过度复极化），最终平稳回到初始的静息电位。

🔬 科学总结：

动作电位是细胞膜电位的一次快速、瞬时变化，由特定离子通道的开闭控制。这一过程对于神经信号的传递和肌肉收缩至关重要，是生命活动中的基础电生理现象。

PS: 神经和肌肉的动作电位的再极化比较快 1-5ms. 心肌约150ms-300ms

PPG

Note

PPG（光电容积描记法）心跳检测的原理，我们需要从两个主要方面来探讨：物理原理和生物学基础。

物理原理 🌟

  1. 光的吸收和反射：PPG技术使用光源（通常是LED）照射皮肤。这些光波在穿过皮肤、组织、血管时会被不同程度地吸收。血液，特别是载氧血红蛋白和还原血红蛋白，对光的吸收特性与其他组织不同。
  2. 检测反射或透射光：皮肤下方的传感器（如光电二极管）用来检测经过组织后反射或透射回来的光。当心脏跳动时，血液量和血液中的血红蛋白浓度会周期性变化，这导致反射或透射光的强度也随之变化。
  3. 信号处理：收集到的光信号经过处理，可以得到血液流动和心脏跳动的相关信息。通常这些信号会显示为波形图，每个波峰对应一次心跳。

生物学基础 🌱

  1. 血流动态：心脏收缩时，血液被泵入动脉，增加了血管中的血液量和压力，这会导致更多的光被吸收。心脏舒张时，情况相反。
  2. 血红蛋白的角色：血液中的血红蛋白是主要的光吸收媒介。其吸收光的能力取决于其是否与氧结合（载氧血红蛋白和还原血红蛋白有不同的光吸收特性）。

📈 应用举例

想象一下，你戴上了一款智能手表。当手表的LED灯照射你的手腕时，不同波长的光被血红蛋白吸收或反射。手表上的传感器检测这些变化，并通过算法转化为心跳数据。每次心脏跳动，血液流量和其特性的微小变化都会被捕捉并记录下来。

🔬 总结：PPG心跳检测通过分析皮肤下血液量的微小变化来监测心跳。这一过程涉及光学物理和生理学，是一种非侵入性、便捷的心率监测方法。

ECG

Note

ECG（心电图）信号的波形组成是基于心脏电生理学的原理。心脏的电活动产生特定的波形，这些波形代表心脏各个部分的激动和复极过程。下面是ECG波形的主要组成部分和它们所代表的心脏活动：

1. P波 🌊

  • 代表：心房激动（心房收缩）
  • 发生原因：由于窦房结（SA结）产生的电脉冲，引起心房肌肉的激动。
  • 特征：P波是ECG的第一个小波，通常是圆顶形的，向上的。

2. PR间期 📏

  • 代表：从心房激动到心室激动的传导时间。
  • 发生原因：电脉冲从窦房结通过房室结（AV结）向心室传导的时间。
  • 特征：PR间期是P波结束到QRS复合波开始之间的直线部分。

3. QRS复合波 📈

  • 代表：心室激动（心室收缩）
  • 发生原因：电脉冲通过心室肌肉，导致心室收缩。
  • 特征：
    • Q波：第一个负向波（如果存在）
    • R波：正向波，通常是QRS复合波中最高的波
    • S波：R波后的负向波

4. ST段 📏

  • 代表：心室激动结束和复极开始之间的期间。
  • 发生原因：心室肌肉激动完成，准备开始复极。
  • 特征：QRS复合波结束到T波开始之间的直线部分。

5. T波 🌈

  • 代表：心室复极（心室放松）
  • 发生原因：心室肌肉电位回归到静息状态。
  • 特征：通常是一个正向波，比P波更宽和更圆。

6. U波（在某些情况下出现）🔍

  • 代表：可能与心室肌肉的后复极有关。
  • 特征：在T波之后出现的一个小波，不总是出现。

🖼️ 波形图解析：

想象一张心电图，上面首先出现的是小巧的P波，随后是平稳的PR间期。接下来是突出的QRS复合波，之后是平直的ST段，最后是圆润的T波。在某些情况下，还可能在T波后看到一个小小的U波。

🔬 总结：ECG波形的每一个部分都揭示了心脏在特定时刻的电生理状态，是评估心脏功能的重要工具。通过理解这些波形，医生可以诊断各种心脏病，如心律不齐、心肌缺血和电解质失衡等。

主要指标

  • @ 心率（Heart Rate
    • 定义：每分钟心跳的次数。
    • 计算方法：通常通过测量R波之间的间隔（R-R间期）计算得出。
    • 正常范围：成人静息心率一般在60到100次/分钟。
  • @ 心律（Rhythm）
    • 定义：心跳的规律性。
    • 评估：观察R波之间的间隔是否均匀。
    • 类型：可分为规律（正常心律）和不规律（如心律不齐）。
  • @ P波
    • 相关性：与心房激动相关。
    • 正常特征：持续时间 <0.12秒，幅度 <2.5毫伏。
    • 异常特征：P波异常可能指示心房肥大或心房颤动。
  • @ QRS复合波
    • 相关性：与心室激动相关。
    • 正常特征：持续时间0.06至0.10秒。
    • 异常特征：宽QRS波可能表示心室肥大、束支传导阻滞或室性心律。
  • @ PR间期
    • 相关性：心房和心室之间的电传导时间。
    • 正常范围：0.12至0.20秒。
    • 异常情况：延长的PR间期可能意味着房室传导延迟。
  • @ ST段
    • 相关性：心室激动后的状态。
    • 评估：应接近基线水平。
    • 异常特征：抬高或降低的ST段可能指示心肌缺血或心肌梗死。
  • @ T波
    • 相关性：心室复极。
    • 正常特征：与前面的QRS波同向。
    • 异常特征：倒置或异常高的T波可能表示心肌缺血或其他心脏问题。
  • @ QT间期
    • 相关性：心室激动和复极的总时间。
    • 正常范围：根据心率的不同而调整（使用QTc公式校正）。
    • 异常情况：延长或缩短的QT间期可能表示电解质失衡、药物影响或遗传性心脏病。
  • @ 心率变异率(Heart Rate Variation)
    • 关注心跳之间的时间间隔（称为RR间隔）
    • HRV被用作评估个体健康状态的一个指标。较高的HRV通常被认为是心脏和整体健康的标志
    • 低HRV与多种疾病的风险增加有关，包括心脏病、焦虑、抑郁和早亡

EDA

Note

Electrodermal activity (EDA) 是指皮肤电活动的变化，这些变化主要是由皮肤的汗腺活动引起的。EDA是测量生理唤醒、情绪反应和心理状态的一种非常敏感的方法。EDA的两个主要分量是皮肤电导（skin conductance）和皮肤电位（skin potential）。

皮肤电导（Skin Conductance）

  • 概念：量度皮肤的导电性，通常表现为对电流的阻力降低。
  • 测量：通常通过电极放置在手掌或手指上来测量。
  • 指标：
    • 皮肤电导水平（SCL）：基线皮肤电导。
    • 皮肤电导反应（SCR）：对特定刺激的快速变化。

皮肤电位（Skin Potential）

  • 概念：皮肤表面的电位差异。
  • 来源：由皮肤下汗腺周围的活动产生。

EDA的生理基础

  • 汗腺活动：EDA的主要驱动因素是皮肤的汗腺活动。情绪唤醒、压力或刺激都可以增加汗腺分泌，从而改变皮肤的电性能。
  • 自主神经系统：这种汗腺活动主要由交感神经系统控制，这是自主神经系统的一部分。

应用

  • 心理学研究：EDA被广泛用于情绪研究、压力反应和心理唤醒的研究。
  • 临床诊断：如焦虑障碍、恐惧症和其他相关疾病的诊断。
  • 法医学：在测谎仪测试中作为反应指标之一。
  • 用户体验研究：在评估产品或系统的情绪反应方面。

重要注意事项

  • 环境影响：温度和湿度等环境因素可以影响EDA测量。
  • 个体差异：不同人的EDA反应可能有显著差异。

📈 总结：

EDA是一种反映皮肤电活动的测量，能够敏感地反映个体的情绪唤醒和心理状态。通过监测皮肤电导和皮肤电位的变化，EDA可以为心理学、临床诊断和用户体验研究提供重要数据。由于EDA受多种因素影响，因此在分析和解释数据时需要谨慎。

信号处理

  1. 信号获取：
    • 通过各种传感器和设备收集生物信号，如心电图（ECG）、脑电图（EEG）、肌电图（EMG）等。
  2. 预处理：
    • 滤波：去除噪声，例如使用低通、高通、带通滤波器。
    • 去除伪迹：去除由运动、电源线干扰等引起的干扰信号。
  3. 信号分割：
    • 将连续的信号分割成可分析的小段，特别是在研究周期性生物信号（如心跳）时。
  4. 特征提取：
    • 从信号中提取有意义的特征，如振幅、频率、时长等。
  5. 特征选择：
    • 确定最有助于分析目标的特征。
  6. 分类/建模：
    • 使用统计或机器学习方法来分类或预测生物状态。
  7. 解释和分析：
    • 根据生理学和临床知识解释结果。
