脑科学总览

#脑科学

大脑(拉丁语: Cerebrum)是人体最复杂和最不可思议的器官之一，它负责控制我们的思想、感觉、行为和身体功能, 是中枢神经系统的最上层的部分. 结构上由左右两个大脑半球组成, 一般分为额叶、顶叶、颞叶和枕叶四个脑叶1。将两个大脑半球隔开的是被称作为“中央纵裂”的沟壑，两个半球除了胼胝体相连以外完全左右分开

主要功能:

  1. 认知功能: 包括记忆、学习、判断和解决问题。
  2. 感觉和感知: 如视觉、听觉、嗅觉、味觉和触觉。
  3. 运动控制: 控制身体的运动和协调。
  4. 情感和社交: 负责情感反应和社交互动

人类对大脑的认识主要通过一下途径:

  1. 神经科学: 通过研究神经元和神经网络来了解大脑的工作机制。
  2. 心理学: 研究大脑如何影响行为和心理状态。
  3. 医学成像: 如MRI和PET扫描用于观察大脑结构和功能。
  4. 人工智能和计算模型: 用于模拟大脑的某些功能，以更好地理解其工作原理。

当前的局限性:

  1. 复杂性: 大脑的复杂性远超过任何现有的计算模型或理论。
  2. 未知领域: 尽管有大量的研究，但关于意识、自由意志等方面的问题仍然是未解之谜。
  3. 疾病和障碍: 对于许多神经性疾病和心理障碍，如阿尔茨海默病、精神分裂症等，人们仍然没有找到有效的治疗方法。

脑解剖学
...

大脑的断面分为白质与灰质5(https://zh.wikipedia.org/wiki/%E5%A4%A7%E8%84%91#cite_note-:2-5)。 大脑的灰质是指表层的数厘米厚的被称为“大脑皮层”的结构，是神经细胞（细胞体）聚集的部分，具有六层的构造，含有复杂的回路是思考等活动的中枢5(https://zh.wikipedia.org/wiki/%E5%A4%A7%E8%84%91#cite_note-:2-5)。相对大脑皮层的灰质，大脑白质又称为大脑髓质

大脑功能和分区
...

1. 大脑皮层（Cerebral Cortex）
...

前额叶（Frontal Lobe）
...

  • 位置: 位于大脑的前部，紧邻额骨
  • 功能: 负责高级认知功能，如思考、规划、决策和行为控制。
  • 特点: 这是人类大脑中最发达的部分，与人类的复杂行为和社会互动有关。

顶叶（Parietal Lobe）
...

  • 位置: 位于大脑的上部和中部，紧邻顶骨
  • 功能: 主要负责空间感知、方向感和数学计算。
  • 特点: 与身体感觉有关，如触觉、温度和疼痛。

颞叶（Temporal Lobe）
...

位置:位于大脑的两侧，紧邻颞骨
功能: 主要负责听觉和语言处理。
特点: 也与记忆和情感有关。

枕叶（Occipital Lobe）
...

  • 位置: 位于大脑的后部，紧邻枕骨
  • 功能: 主要负责视觉处理。
  • 特点: 是视觉信号的主要处理中心。

2. 边缘系统（Limbic System）
...

海马体（Hippocampus）
...

  • 位置: 位于大脑的内部，靠近底部
  • 功能: 负责长期记忆的存储。
  • 特点: 对学习和空间导航也有影响。

杏仁核（Amygdala）
...

  • 位置: 也位于大脑的内部，靠近海马体
  • 功能: 主要负责处理情感反应，如恐惧和愤怒。
  • 特点: 在社交互动和情感记忆中也起作用。

3. 脑干（Brainstem）
...

延髓（Medulla）
...

  • 位置: 位于大脑和脊髓之间的连接处，是脑干的一部分
  • 功能: 控制基本生命维持功能，如呼吸和心跳.
  • 特点: 是大脑和脊髓之间的主要连接点。

脑桥（Pons）
...

  • 位置: 位于延髓上方，连接大脑和小脑
  • 功能: 主要负责连接大脑和小脑，参与运动控制。
  • 特点: 也参与某些自主神经功能，如呼吸。

4. 小脑（Cerebellum）
...

  • 功能: 主要负责运动协调和平衡。
  • 特点: 虽然不直接参与认知功能，但对精细运动控制非常重要。

5. 丘脑
...

丘脑（Thalamus）
...

  • 位置: 位于大脑的下方和后方，紧邻脑干
  • 功能: 作为感觉和运动信号的中继站。
  • 特点: 几乎所有的感觉和运动信号都会经过这里。

下丘脑（Hypothalamus）
...

  • 位置: 位于丘脑下方，靠近脑干
  • 功能: 负责控制体温、饥饿、渴望和其他基本生理需求。
  • 特点: 是内分泌系统的重要组成部分。

脑区间协作
...

这些分区都有特定的位置，并且通过各种神经通路相互连接。这样的布局使得大脑能够高效地执行各种复杂的任务和功能, 以下是脑区间互相协作的例子:

  1. 驾驶汽车
      • 前额叶（Frontal Lobe）: 负责决策和规划，比如何时减速或转弯。
    • 顶叶（Parietal Lobe）: 负责空间感知，帮助你了解车辆与其他物体的相对位置。
    • 枕叶（Occipital Lobe）: 负责处理视觉信息，如看路标和交通灯。
    • 小脑（Cerebellum）: 负责协调手和脚的动作，以控制方向盘和踩油门。
  2. 听音乐并感受情感
    • 颞叶（Temporal Lobe）: 负责听觉，处理音乐的声音。
    • 前额叶（Frontal Lobe）: 负责对音乐进行分析和评价。
    • 杏仁核（Amygdala）: 负责产生对音乐的情感反应
  3. 学习和记忆
    • 海马体（Hippocampus）: 负责存储新信息。
    • 前额叶（Frontal Lobe）: 负责处理和编码这些信息。
    • 丘脑（Thalamus）: 作为信息的中继站，将信息传送到大脑皮层
  4. 运动活动（比如投篮）
    • 前额叶（Frontal Lobe）: 规划和决定投篮的时机和角度。
    • 顶叶（Parietal Lobe）: 计算篮球与篮筐的距离和角度。
    • 小脑（Cerebellum）: 协调手和眼的动作，以准确地投篮。
    • 脑干（Brainstem）: 控制基本的肌肉活动

脑内神经递质
...

神经递质是一种化学物质，用于神经元（神经细胞）之间的信息传递。以下是一些主要的神经递质及其相关信息

多巴胺（Dopamine）
...

  • 功能: 负责愉悦感、奖赏、动机和注意力。
  • 分泌地点: 主要在大脑的中脑区域分泌。
  • 何时分泌: 在体验愉悦或奖赏时，如吃美食、听好音乐等

5-羟色胺（5-Hydroxytryptamine, 5-HT）
...

  • aka. 血清素(Serotonin)
  • 功能: 负责情绪稳定、睡眠质量和食欲。
  • 分泌地点: 主要在大脑和肠道中分泌。
  • 何时分泌: 在感到放松或安全时，也与食物摄取有关

乙酰胆碱（Acetylcholine）
...

  • 功能: 负责学习、记忆和肌肉活动。
  • 分泌地点: 在多个大脑区域和神经-肌肉接头分泌。
  • 何时分泌: 在学习和记忆任务中，以及在肌肉活动时

格ABA（Gamma-Aminobutyric Acid, GABA）
...

  • 功能: 主要的抑制性神经递质，负责减少神经活动。
  • 分泌地点: 在大脑的多个区域分泌。
  • 何时分泌: 在需要减少焦虑或压力时

谷氨酸（Glutamate）
...

  • 功能: 主要的兴奋性神经递质，负责增加神经活动。
  • 分泌地点: 在大脑的多个区域分泌。
  • 何时分泌: 在需要高度注意或反应时。

内啡肽（Endorphins）
...

  • 功能: 负责减轻疼痛和产生愉悦感。
  • 分泌地点: 在下丘脑和脑垂体分泌。
  • 何时分泌: 在体验疼痛或压力时，如运动或受伤

脑内神经递质协作
...

这些神经递质在不同的大脑区域和不同的情境下起到关键作用，它们相互作用和协调，以支持大脑的复杂功能。以下是神经递质相互协作完成任务的例子:

  1. 在吃美食或进行体育活动时，多巴胺和内啡肽通常会同时分泌，共同产生一种愉悦和满足感。多巴胺（Dopamine）: 负责产生愉悦感。内啡肽（Endorphins）: 增强愉悦感，减轻疼痛
  2. 在学习新信息时，乙酰胆碱和谷氨酸会共同作用，以促进记忆的形成和存储。乙酰胆碱（Acetylcholine）: 负责记忆编码. 谷氨酸（Glutamate）: 负责神经可塑性，有助于记忆的形成。
  3. 在压力或焦虑情况下，5-羟色胺和GABA可能会共同作用，以减轻焦虑并促进放松。5-羟色胺（Serotonin）: 负责情绪稳定 GABA（Gamma-Aminobutyric Acid）: 负责减少神经活动，产生放松感
  4. 在需要高度集中注意力的任务中，多巴胺和乙酰胆碱会共同作用，以提高注意力和信息处理能力. 多巴胺（Dopamine）: 负责注意力和集中。乙酰胆碱（Acetylcholine）: 负责警觉和信息处理

脑机接口(BCI) 是一种技术，它使大脑可以直接与外部设备或计算机系统进行交互。这种交互通常不依赖于传统的神经肌肉通道，而是直接通过大脑的神经活动来实现。

  1. 在信号采集方面无法兼顾精度(极细电极, fMRI, 光遗传), 规模(10-20导联精度差, 最多同时监测几千个神经元)和无创(马斯克的脑科学手术机器人和Synchron的微创脑际借口)的三个指标
  2. 在信号处理方面, 大脑信号各种相互关联firing, 处理分析难度大; 其次神经活动的个体差异性大, 使得BCI的开发更加有挑战性
