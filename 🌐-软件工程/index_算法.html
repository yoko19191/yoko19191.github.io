什么是NP完全问题?

P, NP, CoNP, NP hard and NP complete | Complexity Classes - GeeksforGeeks

Tip

在计算理论中，问题通常被分类为决策问题（decision problem）和非决策问题（non-decision problem）。

  1. 决策问题（Decision Problem）: 这类问题的答案是"yes"或"no"。也就是说，这类问题是关于判断某种性质是否存在的问题。例如，给定一个图和两个节点，我们可以提出一个决策问题：“是否存在一条从一个节点到另一个节点的路径？”这个问题的答案只能是"yes"（存在路径）或"no"（不存在路径）。
  2. 非决策问题（Non-Decision Problem）: 这类问题的答案不仅仅是"yes"或"no"，它可能需要更复杂的答案。例如，给定一组数字和一个目标值，我们可以提出一个非决策问题：“找出这组数字中和为目标值的所有数字组合是什么？”这个问题的答案是一个或多个数字组合，而不仅仅是"yes"或"no"。

在计算理论中，我们通常首先考虑决策问题，因为它们更易于分析。然后，我们可以将非决策问题转化为一系列的决策问题来解决。例如，我们可以将上述的非决策问题转化为一系列的决策问题：“是否存在一个数字组合，其和为目标值？”然后，我们可以通过改变目标值来找到所有的解。

P问题(Polynomial), NP问题的子集, 是指我们在多项式时间内中解决的决策问题. P问题在理论和实际中都有可行解(tractable). 比如, 我们可以在多项式时间内使用Dijkstra算法找到一张图中的最短路径.

NP问题(Non-deterministic ~) 是指我们只能在多项式时间内验证问题解, 却不能给出可行解的问题.

NP难问题(NP-Hard) , NP问题的补集, 是可能在多项式时间内有解, 也可能没有, 我们不知道. 所以NP-Hard问题至少和最难的NP问题一样难.

NP-Hard问题并不一定是NP问题, 但如果一个问题即是NP-Hard, 也是NP问题, 那么这个问题就是NP完全问题(NP-Complete)**.

我们现在已经找到了很多NPC 问题. 比如, 旅行商问题(TSP)、背包问题、图着色问题(Graph Coloring)、布尔可满足性（SAT)、最大团（Max Clique）等等. 这些问题都在包括物流、调度、网络设计、生物信息学的广泛领域有实际的价值.

研究NPC问题之所以重要除了契合实际应用场景之外, 还是因为如果我们找到了任意一个NPC问题有多项式时间解, 那么就可以找到所有NP问题的多项式时间解决方案, 即NP问题可以归约为P问题, P=NP(). 反之, 如果我们能证明任意一个NPC问题没有多项式时间内的解, P≠NP. (Cook-Levin定理)

"P是否等于NP"被认为是计算理论中最重要的未解决问题之一. 作为七个"千禧年大奖问题", 克莱数学研究所提供了一百万美元的奖金。

千禧年大奖问题是由克莱数学研究所（Clay Mathematics Institute）在2000年提出的七个未解决的数学问题。每个问题的解答都有一百万美元的奖金。这七个问题包括：

  1. P vs NP问题：这是计算理论中的一个基本问题，关于问题的复杂性和计算机的解决能力。简单来说，P是可以在多项式时间内解决的问题的集合，NP是可以在多项式时间内验证解的问题的集合。P是否等于NP，是一个未解决的问题。

  2. 黎曼猜想：这是一个关于复数域上的黎曼ζ函数零点分布的问题，它是数论中的一个中心问题。

  3. 杨-米尔斯存在性和质量间隙：这是一个关于量子场论的问题，特别是关于弱相互作用和强相互作用的理论。

  4. 黎曼流形正质量定理：这是一个关于广义相对论和微分几何的问题，关于在给定条件下空间的形状和大小。

  5. 海涅猜想：这是一个关于代数几何的问题，关于曲线、曲面和更高维度对象的数学描述。

  6. 毕达哥拉斯定理：这是一个关于数论的问题，关于整数的性质和行为。 (2003, 格雷戈里·佩雷尔曼)

  7. Navier-Stokes存在性和光滑性：这是一个关于流体动力学的问题，关于空气和水等流体的行为。

Note

Cook-Levin定理，也被称为NP完全性定理，是计算机科学中的一个重要定理，由Stephen Cook和Leonid Levin独立提出。这个定理的主要内容是：存在一种特殊的问题，被称为"布尔可满足性问题"（Boolean satisfiability problem，简称SAT问题），如果这个问题能在多项式时间内解决，那么所有的NP问题都可以在多项式时间内解决。

布尔可满足性问题是这样一个问题：给定一个布尔表达式，其中包含一些布尔变量和布尔运算（如与、或、非），是否存在一种变量取值方式使得整个布尔表达式为真？

Cook-Levin定理的重要性在于，它是第一个明确提出NP完全性概念的定理，为后续的NP完全性理论研究奠定了基础。根据这个定理，如果我们能找到一个有效的算法来解决布尔可满足性问题，那么我们就可以用同样的方法来解决所有的NP问题。换句话说，如果我们能证明P=NP，那么就可以证明存在一种多项式时间的算法可以解决所有的NP问题。然而，至今为止，这个问题仍然没有被解决。

需要注意的是，尽管布尔可满足性问题是第一个被证明为NP完全的问题，但现在已经有许多其他问题也被证明为NP完全，例如旅行商问题、图着色问题等。这些问题都有一个共同的特性，即它们都可以在多项式时间内转化为布尔可满足性问题

一些NPC问题:

  1. 旅行商问题（Travelling Salesman Problem, TSP）：假设你是一个旅行商，你需要访问n个城市，每个城市只能访问一次，然后返回出发城市。所有城市之间的距离都已知。问题是，你应该选择什么样的路线，使得总的旅行距离最短？
  2. 背包问题（Knapsack Problem）：假设你有一个背包和一些物品。每个物品都有一定的价值和重量。背包的载重量是有限的。问题是，你应该选择哪些物品装入背包，使得背包中物品的总价值最大，同时不超过背包的载重量？
  3. 图着色问题（Graph Coloring Problem）：假设你有一个图，图中的每个节点都需要被染色，而且相邻的节点不能被染成相同的颜色。问题是，你需要至少多少种颜色，才能完成这个任务？
  4. 布尔可满足性问题（Boolean Satisfiability Problem, SAT）：假设你有一个布尔表达式，它由一些布尔变量和逻辑运算符（如AND, OR, NOT）组成。问题是，是否存在一种赋值方式，使得这个布尔表达式为真？
  5. 最大团问题（Max Clique Problem）：在图论中，团是一个图的子集，其中的每两个节点都是相邻的。问题是，给定一个图，如何找到最大的团，也就是包含最多节点的团？

Van der Waerden定理 的意思是：有 K长的数列和 C 种颜色，无论你用何种方式对 K 长数列着色，你总是可以在足够长的数列中找到 K长的等差数列用同一个颜色着色。

Q: 什么是最优子结构?

A: 最优子结构是动态规划算法中的一个重要概念。一个问题如果具有最优子结构性质，那么意味着其最优解可以通过其子问题的最优解来构造。

具体来说，如果一个问题的最优解包含了其子问题的最优解，那么我们就说这个问题具有最优子结构。这意味着我们可以通过解决子问题，然后将子问题的解组合起来，得到原问题的解。这是动态规划算法的基础。

例如，假设我们要找出一段路径上的最短路径，这个问题就具有最优子结构，因为一段路径的最短路径包含了其任何子路径的最短路径。

最优子结构性质使得我们可以使用动态规划算法来解决问题。在动态规划算法中，我们首先解决子问题，然后根据子问题的解来解决原问题。这种方法可以避免重复解决同一个子问题，从而提高了算法的效率。
